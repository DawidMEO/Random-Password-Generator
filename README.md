Описание

RОписание

Random Password Generator — это десктопное приложение с графическим интерфейсом для генерации надёжных случайных паролей с настраиваемыми параметрами.

Приложение позволяет:

    настраивать длину пароля (8–32 символа);

    выбирать типы символов для генерации (буквы, цифры, спецсимволы);

    генерировать пароли одним кликом;

    копировать сгенерированные пароли в буфер обмена;

    просматривать историю последних 20 сгенерированных паролей.

История паролей сохраняется в JSON‑файле между запусками приложения.
Функциональность

    Ползунок длины пароля — настройка длины от 8 до 32 символов.

    Чекбоксы выбора символов — включение/выключение букв (a‑z, A‑Z), цифр (0‑9) и спецсимволов (!@#$%&*).

    Кнопка генерации — создание нового пароля с текущими настройками.

    Поле вывода пароля — отображение сгенерированного пароля и возможность скопировать его в буфер обмена.

    Таблица истории — просмотр последних 20 паролей с информацией о длине, типах использованных символов и времени создания.

Технические детали

    Язык программирования: Python 3.x.

    GUI‑библиотека: tkinter (встроенная в Python).

    Хранение данных: JSON‑файл (password_history.json).

    Генерация случайных значений: стандартная библиотека random.

Установка и запуск
Требования

    Установленный Python 3.6 или выше.
    Использование

    Запустите приложение командой python main.py.

    Настройте параметры пароля:

        переместите ползунок, чтобы выбрать длину пароля;

        отметьте чекбоксы для типов символов, которые должны присутствовать в пароле.

    Нажмите кнопку «Сгенерировать пароль».

    Сгенерированный пароль появится в поле вывода.

    Нажмите «Копировать в буфер», чтобы скопировать пароль в буфер обмена.

    Просмотрите историю ранее сгенерированных паролей в таблице внизу окна.

Примеры использования (тесты)

Тест 1. Генерация стандартного пароля

    Настройки: длина 12, буквы и цифры включены, спецсимволы выключены.

    Ожидаемый результат: пароль из 12 букв и цифр, например, K7mN9pQ2rX5v.

Тест 2. Генерация сложного пароля

    Настройки: длина 16, все типы символов включены.

    Ожидаемый результат: пароль из 16 символов, включающий буквы, цифры и спецсимволы, например, mP9#kL2@qR7$vN4%.

Тест 3. Проверка валидации длины

    Настройки: установите ползунок на 7.

    Действие: нажмите «Сгенерировать пароль».

    Ожидаемый результат: появляется сообщение об ошибке «Минимальная длина пароля — 8 символов».

Тест 4. Проверка выбора символов

    Настройки: отключите все чекбоксы.

    Действие: нажмите «Сгенерировать пароль».

    Ожидаемый результат: появляется сообщение «Выберите хотя бы один тип символов».

Тест 5. Сохранение и загрузка истории

    Действие: сгенерируйте несколько паролей, закройте приложение, запустите снова.

    Ожидаемый результат: таблица истории отображает последние пароли, сгенерированные в предыдущем сеансе.
